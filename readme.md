# DevCamper API

> Backend API for DevCamper application, a bootcamp directory

## Usage

Rename "config/config.env.emv" to "config/config.env" and update the values/settings to your own.

## Install Dependencies

```
yarn add
```

## Run App

```
# Run in dev mode
yarn dev

# Run in prod mode
yarn start
```
